# Bay Park Ollama Console

Bay Park Ollama Console is a self-hosted Python Network Assistant and interactive
text-adventure platform.

The project combines:

- Python web development
- Linux and Raspberry Pi hosting
- Git and GitHub-based updates
- A graphical Tkinter adventure editor called MaaazeRunner
- Generated rooms, items, puzzles, and known answers
- JSON and Python validation
- systemd services and scheduled publishing
- Backups, checksums, rollback procedures, and protected application releases

The project was created as a practical exercise in programming, Linux system
administration, automation, troubleshooting, documentation, and
cybersecurity-minded change control.
